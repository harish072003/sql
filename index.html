SQL Tutorial

Introduction

Database is a collection of data. Structured Query Language (SQL) lets you perform CRUD operations: Create, Read, Update, or Delete operations on a database.

  • DBMS: Software interface responsible for authentication, concurrency, logging, and optimization.
  • RDBMS Examples: MySQL, SQL Server, Oracle, PostgreSQL.

Data Types

Character Data

  • char: Fixed-length string. Example: char(5)
  • varchar: Variable-length string. Example: varchar(5)

Numerical Data

  • Whole Numbers: tinyint, int, bigint
  • Decimal Numbers: float, double, decimal

Temporal Data

  • Date, Time, Timestamp

Basic Queries

Create and Drop Database

CREATE DATABASE Harish;
DROP DATABASE Harish;
SHOW DATABASES;

Create Table

                CREATE TABLE harish (
                    EMPNO INT PRIMARY KEY,
                    ENAME VARCHAR(50),
                    JOB VARCHAR(50),
                    MGR INT,
                    HIREDATE varchar(50),
                    SAL DECIMAL(10, 2),
                    COMM DECIMAL(10, 2),
                    DEPTNO INT
                );

                DROP TABLE student; -- drops the table

DESCRIBE student; -- describes the columns in the table student

ALTER TABLE student ADD department VARCHAR(5); -- Adds a new column department to the student table

ALTER TABLE student DROP COLUMN department; -- drops the department column from student table
-- ---or---
ALTER TABLE student DROP department; -- same as above

                select*from harish;
                alter table harish change SAL SALARY DECIMAL(10, 2);
                SELECT *FROM harish;
                Alter table harish drop column AGE;

                Alter table harish RENAME COLUMN COMM to COM;

DML

                select * from harish where JOB='MANAGER';
                select*from  harish where JOB='MANAGER' or JOB='ANALYST';
                select * from harish where JOB in ('MANAGER', 'ANALYST');
               
                 select * from harish where JOB Not in ('MANAGER', 'ANALYST');
               select * from harish where SAL between 800 AND 1100;
                alter table harish add column AGE int;
                alter table harish  drop column AGE;
                INSERT INTO harish(EMPNO,ENAME) values(200,'harish');
               select * from harish where SAL>200 limit 5;
               update harish set AGE=20 where JOB='MANAGER';
               select distinct JOB from harish;

Functions

  • COUNT(): Counts rows.
  • AVG(): Calculates average.
  • MAX(), MIN(): Returns maximum or minimum value.
SELECT COUNT(*) FROM employee;
SELECT AVG(salary) FROM employee WHERE job_desc="MANAGER";

Group By and Having

Group data by column and apply aggregate functions.

SELECT job_desc, COUNT(emp_id) AS count
FROM employee
GROUP BY job_desc
HAVING COUNT(emp_id) > 1;
